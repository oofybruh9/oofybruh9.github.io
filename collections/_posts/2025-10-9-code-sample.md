## list dir in python

```python
import os

print("hello world")
os.system("ls")

# output: hello world
# output: sample_dir

```
## list dir in the command prompt

```shell
ls

```

## list dir in Windows CMD

```bat
@echo off

echo "hello world"
dir

```

## list dir in bash

```bash
#! /bin/bash

echo "hello world"
ls

```
